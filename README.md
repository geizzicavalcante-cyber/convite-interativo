Convite Interativo – Doce Mania Confeitaria Artesanal
Descrição

Este projeto consiste em um convite interativo online, desenvolvido para aniversários e eventos, focando em experiências visuais e auditivas. Ele permite que o usuário:

Abra o convite clicando no botão inicial 💌

Ouça música de fundo durante a visualização 🎵

Explore seções interativas, como Sugestões de Presentes, com imagens e informações detalhadas 🎁

Navegue entre telas de forma animada, proporcionando uma experiência moderna e dinâmica ✨

⚠️ Aviso: Este projeto está em fase inicial e em desenvolvimento. Ainda existem muitas funcionalidades e melhorias planejadas, como animações mais elaboradas, integração com redes sociais e personalizações adicionais.

Diferenciais do Projeto

✔️ Botão “Abrir Convite”

✔️ Animações simples e elegantes

✔️ Sistema de tema personalizável usando variáveis CSS

✔️ Layout responsivo, funcionando em desktops e dispositivos móveis

✔️ Botão WhatsApp funcional para confirmação de presença

Fluxo Interativo

O usuário abre o convite

Pode voltar para a tela inicial

Pode abrir o convite novamente quantas vezes desejar

Interações possuem animações suaves para melhorar a experiência

Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilos, animações e temas

JavaScript – Interatividade e controle do fluxo do convite

Funcionalidades

Tela inicial com botão “Abrir Convite”

Animação suave de transição entre telas

Botão de Localização com link para Google Maps 📍

Confirmação de presença via WhatsApp 💬

Página de Sugestão de Presentes, com cards interativos

Layout responsivo e estilizado
